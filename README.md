# ExerciciosHeranca

# Como executar:

## Crie a pasta build (se não existir)
```mkdir build```

Utilizando o **WLS** Na pasta **build** do projeto execute os seguintes comandos:

## Primeiro rode o Cmake:
```cmake ..```

## Após isso, rode o make:
```make```

## Por fim execute com o comando:
```./seu_executavel```
